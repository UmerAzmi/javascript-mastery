Topic 12: String Methods

Check the console to see string manipulation examples with "Umer Azmi" and "123-456-7890"

Available String Methods:

  • .length - Get string length
  • .charAt(index) - Get character at position
  • .indexOf(char) - Find first occurrence
  • .lastIndexOf(char) - Find last occurrence
  • .trim() - Remove whitespace
  • .toUpperCase() - Convert to uppercase
  • .toLowerCase() - Convert to lowercase
  • .repeat(count) - Repeat string
  • .startsWith() - Check if starts with
  • .endsWith() - Check if ends with
  • .includes() - Check if contains
  • .replaceAll(old, new) - Replace all occurrences
  • .padStart(length, char) - Pad at beginning
  • .padEnd(length, char) - Pad at end
