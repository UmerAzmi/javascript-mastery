Topic 48: querySelector vs querySelectorAll

Two methods to select elements using CSS selectors

Feature querySelector() querySelectorAll()
Returns FIRST matching element ALL matching elements (NodeList)
If no match null Empty NodeList []
Use case Single element (usually ID) Multiple elements (class, tag)
Can loop? No (single element) Yes (forEach, for loop)
// querySelector - Returns ONE element const header = document.querySelector("#header"); // ID const firstBox = document.querySelector(".box"); // Class // querySelectorAll - Returns ALL elements const allBoxes = document.querySelectorAll(".box"); allBoxes.forEach(box => { box.style.color = "red"; // Apply to each });
CSS Selectors work: .class, #id, tag, [attribute], parent > child, etc.
